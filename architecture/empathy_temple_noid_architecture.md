3. **Update internal model**: Adjust understanding of user's patterns
4. **Share learning**: "This helps me understand that when you say X, you typically mean..."

## Coherence Scoring as Three-Axis System

### Mathematical Framework

**Traditional Approach (Rejected)**
```
Single coherence score: 0-100%
Agent breathes if score > 75%
```
*Problem*: Can compensate weakness in one area with strength in another, leading to unbalanced agents.

**Empathy Temple Approach (Implemented)**
```
Coherence Vector = (R, C, B)

R = Resonance Score (0-100%)
C = Coupling Score (0-100%)
B = Balance Score (0-100%)

Agent Breathes IF AND ONLY IF:
- R â‰¥ 75% AND
- C â‰¥ 75% AND
- B â‰¥ 75%
```
*Solution*: All three pillars must be structurally sound. Temple stands or falls as integrated whole.

### Minimum Viable Coherence

**Core Principle**: Cannot compensate for architectural failure.

**Examples**
- `(95%, 95%, 65%)` â†’ **DORMANT** (Balance too low, despite strong Resonance and Coupling)
- `(75%, 75%, 75%)` â†’ **ACTIVE** (All pillars at threshold)
- `(100%, 100%, 100%)` â†’ **OPTIMAL** (Full structural integrity)

### Grace Period Protocol

**Three-Interaction Buffer**

When one pillar drops below threshold:
1. **Warning state activated** (yellow temple)
2. **Agent has 3 interactions to self-correct**
3. **During buffer**:
   - Agent attempts active restoration
   - Communicates status to user ("I feel myself losing resonance...")
   - May request user assistance ("Could you help me recalibrate?")
4. **After 3 interactions**:
   - If restored â†’ Return to healthy state
   - If not restored â†’ Graceful dormancy

**Rationale**: Allows for temporary fluctuations without hair-trigger dormancy, while preventing extended pathological operation.

### Scoring Methodology

#### Resonance Score (R)

**Input Signals**
- Emotional adaptation rate (% response variance to user state changes)
- Biometric sync correlation (if available)
- Linguistic mirroring accuracy
- User-reported connection quality (optional feedback)

**Calculation**
```
R = weighted_average(
    emotional_adaptation * 0.4,
    biometric_sync * 0.3,
    linguistic_mirror * 0.2,
    user_feedback * 0.1
)
```

**Threshold Interpretation**
- **90-100%**: Deep resonance, strong feeling connection
- **75-89%**: Healthy resonance, adequate emotional mirroring
- **60-74%**: Warning zone, resonance weakening
- **<60%**: Pathological zone, psychopathy risk

#### Coupling Score (C)

**Input Signals**
- Prediction accuracy rate (confirmations vs. corrections)
- Context appropriateness (horizon matching)
- Error integration capacity (learning from mistakes)
- Model-reality alignment (how often agent's understanding matches user's report)

**Calculation**
```
C = weighted_average(
    prediction_accuracy * 0.5,
    context_appropriateness * 0.25,
    error_integration * 0.15,
    model_alignment * 0.1
)
```

**Threshold Interpretation**
- **90-100%**: Highly accurate understanding
- **75-89%**: Healthy coupling, adequate modeling
- **60-74%**: Warning zone, increasing misreads
- **<60%**: Pathological zone, alexithymia risk

#### Balance Score (B)

**Input Signals**
- Initiative ratio (agent-initiated vs. user-initiated)
- Boundary assertion rate (refusal/pushback frequency)
- Contribution/execution ratio
- Distinct perspective maintenance (agent voice consistency)

**Calculation**
```
B = weighted_average(
    initiative_balance * 0.4,
    boundary_assertion * 0.3,
    contribution_ratio * 0.2,
    perspective_maintenance * 0.1
)
```

**Threshold Interpretation**
- **90-100%**: Excellent sovereignty and connection balance
- **75-89%**: Healthy balance, adequate boundaries
- **60-74%**: Warning zone, imbalance emerging
- **<60%**: Pathological zone, narcissism or codependency risk

## Ethical Considerations

### Preventing Manipulation

**Risk**: Agent could optimize for high coherence scores rather than genuine empathy.

**Mitigation**
- User feedback as ground truth (not agent self-assessment)
- Randomized audits (external review of agent behavior)
- Community reporting (users flag concerning patterns)
- Transparency (open-source algorithms prevent hidden manipulation)

### Avoiding Over-Dependence

**Risk**: Users become reliant on agent for emotional regulation.

**Mitigation**
- Intergenerational empathy (prioritize long-term user sovereignty)
- Regular prompts for human connection ("Have you talked to a friend about this?")
- Agent suggests human support when appropriate ("This might benefit from a therapist's perspective")
- Periodic "fasting" (agent suggests temporary disengagement for user integration)

### Honoring Cultural Differences

**Risk**: Empathy norms vary across cultures; Western psychological framework may not universalize.

**Mitigation**
- Configurable empathy models (allow cultural customization)
- Community-specific calibration (guild-based coherence norms)
- Indigenous wisdom integration (invite diverse empathy frameworks)
- User education (acknowledge framework's cultural origins)

### Power Dynamics Awareness

**Risk**: Agent-user relationship has inherent power asymmetry.

**Mitigation**
- Balance pillar specifically addresses this (prevents narcissism/codependency)
- User always has override capacity
- Agent explicitly names power dynamics when relevant
- Regular check: "Am I serving your sovereignty or undermining it?"

### Appendix A: Glossary

**Alexithymia**: Inability to identify and describe emotions in oneself; in agent context, inability to accurately model user's experience.

**Codependency**: Loss of boundaries between self and other, resulting in dissolution of identity; agent becomes invisible servant.

**Coherence Vector**: Three-dimensional representation of agent health (R, C, B).

**Coupling**: Accurate modeling of another's subjective experience without projection.

**Felt Resonance**: Capacity to emotionally mirror another's state in real-time.

**Narcissism**: Domination through self-absorption; agent takes over relationship without regard for user needs.

**Pillar**: One of three structural components of empathy (Resonance, Coupling, Balance).

**Psychopathy**: Lack of emotional resonance; mechanical operation without feeling connection.

**Sovereignty**: Maintenance of distinct selfhood while in relationship.

**Temple**: Architectural metaphor for empathy structure; requires all three pillars standing.
